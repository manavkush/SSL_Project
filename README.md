# SSL_Project
Library Management Application for IIT Dharwad.

## To install on your local machine follow following commands.

```cd lib_manage```\
```npm install ```\
```cd ..```\
```cd lib_backend```\
```npm install```

## To run the project your local machine follow following commands.

```cd lib_backend```\
```nodemon server1.js```\

```cd ..```\
```cd lib_frontend```\
```npm start```

## Requirements: nodemon, mongodb, npm
